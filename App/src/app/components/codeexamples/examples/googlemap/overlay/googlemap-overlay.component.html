Google Map API

Chart Themes - at the top you can select Silver, Night, and Normal
Google GeoLocation API - at the top you can search for an address and see that marker show up using the Google API

Markers - Displays a few company HQ markers on the the map with logos from the GoogleMaps API controller on the backend

Find You - If you accept letting the site use your location, will display a user icon on your location and pan to it

Traffic - Displays Traffic Layer from Google Map API

Weather - Displays NOAA wearger KML Layer

Air Quality - Displays Air Quality KML Layer

Fire - Displays Fire KML Layer from USDA Forest Service

Drawing - Allows Map drawing through Google API, I recommend adding the markers and then drawing a circle that encapsulates them for a discription under the map

Also If you use the line and click and then double click you will get a distance calculation in miles below the map

Directions - Allows To and From address. Direction will display on map as well as step by step under the map


API Calls

Google Maps is basically all API calls, with special notes for the GeoLocation stuff. The Markers are also brought in with the API project in the Google Controller
