Polygon.IO

I am using the Polygon.IO API to pull back some financial data on stocks by symbol. Its a really cool API and easy to use, they dont offer the current quote/price for free and I didnt want to pay $199.00 a month for it so sorry this is just for demo purposes. Below are a few example symbols to help if you dont have any handy:

  • TSLA : Tesla

  • AAPL : Apple
